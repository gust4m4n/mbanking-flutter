# mbanking-flutter

Demo mobile banking application menggunakan flutter, lengkap dengan UI, mock api dan contracts.

Fitur yang tersedia:
+ Login / Logout
+ PIN dan Biometric Login
+ Ganti theme/warna
+ Transfer Antar Rekening
+ Transfer Antar Bank
+ Tarik Tunai
+ QRIS
